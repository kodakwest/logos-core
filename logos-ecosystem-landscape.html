Principles

  1. No deep integration — each app stands on its own. The hub links to them.
  2. Bible data from one source — LogOS Core D1 DB is the single source of truth for scripture.
  3. Sermon data from one source (future) — plan unified sermon content model.
  4. Shepherd is ready now — links go live immediately from the hub.
  5. Roundtable comes later — Coming Soon until ready.
